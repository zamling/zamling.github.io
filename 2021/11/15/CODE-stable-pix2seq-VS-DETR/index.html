Transformer

Encoder

Decoder

这里需要改变一个思路，在DETR中，输出的格式为[decoder layer number, batch size, N (the number of queries), C]。而在stable-pix2seq中，N代表的是seq length。在这里，max_seq_length设置为了500，也就是最多检测100个框

他的tgt求和了两次embedding, 一次是queries的，一次是可学习的position embedding

并且，由于是序列而非DETR的并行，所以这里的decoder要类似于Transformer源码，加入seq mask。并在推理的时候，循环推理

只看最后一个layer的最后一个batch输出（最新词的输出）
